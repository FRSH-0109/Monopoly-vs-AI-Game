\doxysubsection{Detailed Description}
Class representing the main game engine for the Monopoly game. 

The \mbox{\hyperlink{class_monopoly_game_engine}{Monopoly\+Game\+Engine}} class handles the overall game flow, including player turns, actions, and interactions with the game board. It also manages the graphical user interface (GUI) components.
